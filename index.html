What Can I Do For You?

Want some thoughts on how I add value to my customers and employers?

Automation

Automation is critical in the contemporary web operations world. Deployments need to be fast, error-resistent, infalliable, and simplified. Automation enhances that process and enables processes to be repeatable and auditable. I consider a drive for automation to be a key motivator and skill.

Infrastructure Management

It can be easy to spin up a private server in AWS or Dreamhost and push out a Wordpress blog. But can that process be expanded and scaled to larger deployments? A flexible fabric of infrastructure allows applications to scale and enhance availability. Managing this scalability and monitoring its effectiveness is another key component of my skill set.

Security

Classically trained in application and network security at my alma mater. Security was a focal point in the development of my career--ensuring that sailors at sea were secure from cyberattacks and that customers at home were safe from the same even when operating at massive scales. Availability and flexibility are key for a modern web application stack but it must be secure in order to ensure that there is an application to serve!
